# P2P Lab1 C++

Лабораторная работа №1 - Работа с пиринговыми сетями в среде .NET (реализация на C++)

## Описание

Реализация двух экспериментов с микросервисами и пиринговыми сетями с использованием Boost.Asio.

### Эксперимент 1
Создание 2-3 микросервисов, подключенных к определенным портам для:
- Передачи/приема сообщений
- Считывания параметров
- Работы с ресурсами

### Эксперимент 2
Поиск пиров в двух режимах:
- **Полностью децентрализованный** - пиры ищут друг друга самостоятельно
- **Гибридный** - с центральной таблицей пиров (как в торрент-трекере)

## Технологии

- C++20
- Boost.Asio (сетевые взаимодействия)
- CMake (сборка)


## Сборка

```bash
mkdir build && cd build
cmake ..
make```
